# Evolution of Agriculture in Mainland Portugal (1989–2019)

## Project overview

This repository contains the final project developed for an agro-environmental data analysis course. The project examines structural changes in agriculture in mainland Portugal between 1989 and 2019, using official data from Statistics Portugal (INE) on agricultural labour, agricultural area, temporary crops, permanent grasslands and pastures, and crop productivity.

The main goal is to build a coherent story from a complex agricultural dataset and to understand how the agricultural sector changed over time in mainland Portugal.

## Research question

How did the structure of agriculture in mainland Portugal change between 1989 and 2019?

## Working hypotheses

- Agricultural labour decreased over time.
- The total agricultural area changed across the reference years.
- The number of agricultural holdings with temporary crops changed over time.
- Permanent grasslands and pastures followed a distinct long-term trend.
- Productivity patterns differed across crop groups.

## Data sources

The analysis is based on data from Statistics Portugal (INE), with a focus on mainland Portugal and the years 1989, 1999, 2009 and 2019.

Main datasets used in the project:

- Agricultural labour
- Agricultural holdings area
- Holdings with temporary crops
- Permanent grasslands and pastures area
- Productivity of main agricultural crops

## Study area and period

The territorial focus of the project is mainland Portugal. The analysis concentrates on four reference years: 1989, 1999, 2009 and 2019.

## Methods

The project follows these main steps:

1. Definition of the research question and working hypotheses.
2. Data collection and preparation.
3. Data cleaning and harmonization across years.
4. Summary statistics.
5. Exploratory data analysis.
6. Inferential analysis, where appropriate.
7. Final visualization and storytelling.
